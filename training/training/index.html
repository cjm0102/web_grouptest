This dashboard provides essential functions to manage participants and ensure the security and privacy of user data.

Navigation Bar

The navigation bar above helps you easily access key sections of the admin dashboard. Use it to navigate to the list of participants, add new participants, or log out securely.

Footer

The footer contains additional information and links that may be relevant to your users. It enhances the overall structure of the dashboard.

The dashboard is styled using Bootstrap, a popular front-end framework. Bootstrap ensures a responsive and visually appealing design, making the user experience more seamless.

Clicking on "Log Out" in the navigation bar allows you to securely end your session. Always log out to ensure the security of the admin dashboard.

Access the "Terms and Service" page for a detailed overview of the terms and conditions governing the use of this admin dashboard. It provides important information about usage policies.

The "Privacy" page outlines how user data is handled and protected within the admin dashboard. It emphasizes our commitment to maintaining the privacy and confidentiality of user information.
